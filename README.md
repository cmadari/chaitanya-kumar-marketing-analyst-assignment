**Cross-Channel Marketing Performance Dashboard**
**Overview**
This project focuses on analyzing advertising performance across multiple platforms by integrating data from Facebook, Google, and TikTok into a unified dataset.
The goal was to standardize key metrics across platforms and build a single source of truth for evaluating campaign performance. Using this unified data model, I developed a dashboard to track key performance indicators such as impressions, clicks, conversions, and spend, along with derived metrics like CTR, CPC, and CPA.
This dashboard enables cross-channel comparison, campaign-level insights, and trend analysis to support data-driven marketing decisions.

**Objectives**
Combine multi-platform advertising data into a unified structure
Standardize key performance metrics across platforms
Analyze campaign performance and engagement
Identify trends and optimization opportunities
Enable better budget allocation decisions

**Tools & Technologies**
SQL / Data Processing – Data preparation and transformation
Python – Data cleaning and standardization
Microsoft Power BI – Dashboard development and visualization
Excel (optional) – Supporting analysis

**Dataset**
The project uses three datasets:
Facebook Ads Data
Google Ads Data
TikTok Ads Data
These were combined into a unified dataset with the following fields:
Date
Campaign
Platform
Impressions
Clicks
Spend
Conversions

**Data Processing Steps**
Loaded raw data from three platforms
Standardized column names and formats
Combined datasets into a unified table
Validated data for consistency and completeness
Created calculated metrics for performance analysis

**Key Metrics**
Impressions – Number of times ads were shown
Clicks – Number of user interactions
Conversions – Desired actions completed
Spend – Total advertising cost

**Derived Metrics:**
CTR (Click-Through Rate) = Clicks / Impressions
CPC (Cost Per Click) = Spend / Clicks
CPA (Cost Per Acquisition) = Spend / Conversions

**Dashboard Features**
KPI summary (Impressions, Clicks, Conversions, Spend, CTR)
Platform-wise performance comparison
Campaign-level analysis
Trend analysis over time
Interactive filtering by platform and date

**Key Insights**
Facebook showed the most cost-efficient conversions with the lowest CPA
Google Ads had the highest CTR, indicating strong engagement
TikTok delivered the highest impressions, making it effective for awareness
