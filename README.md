# chaitanya-kumar-marketing-analyst-assignment
Power Bi Dashboard for the marketing analyst assignment

## Overview
This project integrates Facebook, Google, and TikTok advertising data into a unified dataset and presents cross-channel insights using Tableau.

## Tools Used
- SQL
- Python
- Power Bi Desktop

## Key Insights
- Facebook → Best CPA
- Google → Best CTR & ROAS
- TikTok → Highest reach
